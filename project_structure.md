# Documentation
This is the documentation folder containing .ai_context files.

- `ai_context/`: Contains context files for AI integration.
  - `usage.ai_context`
  - `faq.ai_context`

# Core Modules
- `modules/`: Core functional modules for the Signalmapper project.
  - `module1/`: Description of module1.
  - `module2/`: Description of module2.

# Shell Scripts
- `scripts/`: Shell scripts for automation.
  - `install.sh`: Installation script.
  - `deploy.sh`: Deployment script.

# Configuration Files
- `config/`: Configuration files for different environments.
  - `development.json`: Configuration for development.
  - `production.json`: Configuration for production.

# Development Infrastructure
- `infrastructure/`: Infrastructure setup files.
  - `Dockerfile`: Docker configuration.
  - `docker-compose.yml`: Docker Compose configuration.

# README
- `README.md`: Project overview and instructions.
