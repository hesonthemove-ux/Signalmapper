# SignalMapper Pro - System Manifest
## Core Functional Requirements:
1. **Hardware Handshake:** Live GNSS (/dev/ttyUSB0) & SDR status LEDs (Green/Red).
2. **Dynamic SS-DAB:** Filter sites within +20 miles of Ofcom polygon boundaries (Paisley, Greenock, etc.).
3. **Hybrid Search:** Standard DAB + Manual Frequency Entry (with hardware warnings).
4. **Mission Control:** Purpose selection (Business/Personal) + Satellite/Street/Terrain toggles.
5. **SatNav:** Postcode entry, lane guidance, and TTS (Text-to-Speech) directions.
6. **GIS Reporting:** 100% Core (Green), 20% Buffer (Blue), 40% Overspill (Grey) polygons.
